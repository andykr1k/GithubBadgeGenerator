## Getting Started...

### Clone the code!
```
git clone 
```

### Go to directory
```
cd GithubBadgeGenerator
```

### Set Up Executable
```
make
```

### Start Program
```
./generator.x
```

### Output:

A readme markdown file will be generated where the program is stored.
